
# Phishing Email Detection & Awareness System

## 📌 Project Overview

This project focuses on analyzing phishing email samples and identifying common phishing indicators such as fake sender domains, suspicious links, and urgency-based messages.
The goal of this project is to improve cybersecurity awareness and help users recognize and avoid phishing attacks.

This project was completed as part of a **Cyber Security Internship Task**.

---

## 🎯 Objectives

* Analyze phishing email samples
* Identify phishing indicators
* Classify email risks (Safe / Suspicious / Phishing)
* Explain how phishing attacks work
* Provide prevention and awareness guidelines

---

## 🛠 Tools Used

* Google Admin Toolbox Messageheader (Email Header Analyzer)
* Web Browser for link inspection
* Google Docs / MS Word for documentation
* Public phishing email datasets for study

---

## 📧 Phishing Email Examples Analyzed

1. **Bank Account Verification Scam**
2. **Microsoft Password Reset Phishing Email**
3. **Fake Delivery Notification Email**

Each email was analyzed for suspicious indicators such as:

* Fake or suspicious sender domains
* Urgent or threatening language
* Suspicious links
* Generic greetings

---

## ⚠ Risk Classification

All analyzed emails were classified as:

**High Risk – Phishing**

because they contained multiple phishing indicators including fake domains and suspicious URLs.

---

## 🔎 Email Header Analysis

Email headers were analyzed using an email header analyzer to inspect sender information, mail servers, and routing details.

This helped identify suspicious domains and potential phishing sources.

---

## 🛡 Prevention Guidelines

Users can protect themselves from phishing attacks by following these best practices:

* Always verify the sender’s email address
* Avoid clicking suspicious links
* Check website URLs carefully before entering credentials
* Enable multi-factor authentication (MFA)
* Report suspicious emails to the security team

---

## 📚 Key Phishing Indicators Identified

* Fake or spoofed sender domains
* Urgent language creating panic
* Suspicious URLs or shortened links
* Generic greetings instead of user names
* Requests for sensitive information

---

## 📊 Outcome

Through the analysis of phishing email examples, this project demonstrates how phishing attacks attempt to manipulate users and how individuals can identify and prevent such attacks.

Cybersecurity awareness is one of the most effective defenses against phishing attacks.
